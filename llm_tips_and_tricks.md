## Using LLMs

### Prompt Engineering
Quick Reference: Prompt Formulas

| Situation                     | Prompt Type                  | Key Phrase                                           |
|------------------------------|------------------------------|------------------------------------------------------|
| Code doesn't work            | Debugging                    | "Identify the bug(s) and explain why"                |
| Comparing solutions          | Trade-offs                   | "Pros and cons for each approach"                    |
| Want to improve code         | Optimization                 | "Identify 3 bottlenecks and optimize"                |
| Need to explain to interviewer | Explain Like I'm Interviewing | "Make it sound natural, like I'm explaining to a peer" |
| Code might break             | Edge Cases                   | "Identify 5 edge cases that could break this"        |
| Don't understand concept     | Incremental Learning         | "Explain in simple terms, then build up"             |
| Getting feedback             | Code Review                  | "Act as a senior engineer doing code review"         |
| Setting up the problem       | Constraints-First            | "Production-ready code with comments"                |
